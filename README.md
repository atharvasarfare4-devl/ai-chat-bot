# ⚛️ PHYSICS DADA – AI Science Engine

**PHYSICS DADA** is a specialized AI-powered chat interface designed to simplify complex scientific concepts. Built with a focus on physics and general science, it provides a clean, thematic UI for interacting with AI models.

## 🚀 Features
- **Thematic UI:** Custom-designed interface using a "Vedic-Modern" color palette (Saffron, Turmeric, Indigo).
- **Responsive Design:** Fully optimized for both desktop and mobile viewing.
- **Scientific Focus:** Specifically tailored for students and researchers exploring science and technology.

## 🎨 Design Palette
This project uses a unique color system defined in CSS variables:
- **Saffron & Turmeric:** For highlights and energy.
- **Indigo & Black:** For a deep, focused workspace environment.
- **Space Mono:** Monospaced typography for a technical, "engine" feel.

## 🛠️ Tech Stack
- **HTML5**
- **CSS3** (Custom Properties & Flexbox)
- **Google Fonts** (Space Mono)
